    Just follow the steps to build `param <https://github.com/ioam/param>`_:

    #. Install Python and Visual Studio Community
    #. Get the sources from GitHub
    #. Run ``python setup.py build_ext --inplace``
    #. Run ``python setup.py bdist_wheel``
